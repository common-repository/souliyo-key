== Description ==

An easy-to-use WordPress Plugin to add custom CSS styles that override Plugin and Theme default styles. This plugin is designed to meet the needs of administrators who would like to add their own CSS to their WordPress website.


**Features**

- Simplest way of adding Lao font to your Wordpress website

== Installation ==

Install Souliyo Key For Wordpress just as you would any other WP Plugin:

1.  [Download Souliyo key](http://wordpress.org/plugins/souliyo-key "Souliyo Key") from WordPress.org.

2.  Unzip the .zip file.

3.  Upload the Plugin folder (souliyo-key/) to the wp-content/plugins folder.

4. Go to [Plugins Admin Panel](http://codex.wordpress.org/Administration_Panels#Plugins "Plugins Admin Panel") and find the newly uploaded Plugin, "Souliyo Key For Wordpress" in the list.

5. Click Activate Plugin to activate it.

[More help installing Plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins "WordPress Codex: Installing Plugins")
